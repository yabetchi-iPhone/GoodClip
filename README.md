# GoodToDo

iPhoneだけでChatGPTを使って、iOS向けタスクマネージャー（Web版も）を作るオープンソース・プロジェクトです。

MacやPCを使わずに、モバイルだけでどこまで本格的なアプリ開発ができるか挑戦中！

## コンセプト
- **iPhoneオンリー開発**：ChatGPT + iPhoneのアプリ/ツールだけで完結
- **Minimalist Design**：シンプルで見やすいタスク管理
- **クロスプラットフォーム**：Web（GitHub Pages）でも動くように

## 現在の状態
- HTML + JavaScript で基本的なタスク追加・表示機能を実装
- ChatGPTを活用したプロトタイプ作成
- GitHub Pagesで公開予定

## デモ
→ [Live Demo](https://yabetchi-iphone.github.io/GoodToDo/)

## 使い方
1. リポジトリをClone（またはGitHub Pagesで直接アクセス）
2. `index.html` をブラウザで開く
3. タスクを追加して管理
